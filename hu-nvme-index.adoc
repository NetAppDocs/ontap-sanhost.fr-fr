= Découvrez comment configurer les hôtes SAN ONTAP avec NVMe-oF
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Vous pouvez configurer les hôtes SAN compatibles pour utiliser le protocole NVMe over Fabrics (NVMe-oF) avec ONTAP comme cible de stockage. NVMe-oF inclut NVMe over Fibre Channel (NVMe/FC) et NVMe over TCP (NVMe/TCP). Selon votre système d'exploitation hôte et la version d'ONTAP, vous configurez et validez le protocole NVMe/FC ou NVMe/TCP, ou les deux, sur le système d'exploitation hôte.
