LES utilitaires d'hôtes unifiés Windows (WUHU) sont des programmes logiciels dont la documentation permet de connecter des ordinateurs hôtes aux disques virtuels (LUN) sur un SAN NetApp. NetApp recommande de télécharger et d'installer le dernier kit d'utilitaire. Pour obtenir des informations et des instructions sur la configuration du WUHU, reportez-vous à la section link:https://docs.netapp.com/us-en/ontap-sanhost/hu_wuhu_71_rn.html["Documentation Windows Unified Host Utilities"] Et sélectionnez la procédure d'installation pour votre version de Windows Unified Host Utilities.
