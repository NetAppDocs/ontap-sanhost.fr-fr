Les utilitaires Linux Host vous aident à gérer le stockage ONTAP connecté à un hôte Linux. NetApp recommande fortement d’installer les utilitaires Linux Host, mais ce n’est pas obligatoire. Les utilitaires Linux Host améliorent la gestion et aident le support client NetApp à recueillir des informations sur votre configuration.
