= Découvrez comment configurer les hôtes SAN ONTAP avec FCP et iSCSI
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Vous pouvez configurer certains systèmes d'exploitation hôtes SAN pour FCP et iSCSI avec ONTAP comme cible. Si votre système d'exploitation hôte est pris en charge par le logiciel SAN Host Utilities, vous devez d'abord installer le package utilitaire correspondant au système d'exploitation, qui inclut le kit d'outils SAN. Ensuite, pour tous les types de systèmes d'exploitation hôtes, vous configurez les paramètres de multipath pour les LUN ONTAP.
