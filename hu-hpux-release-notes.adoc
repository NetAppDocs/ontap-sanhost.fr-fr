== Nouveautés de HP-UX Host Utilities 6.0

Il n'y a pas de nouvelles fonctionnalités ni d'améliorations.

HP-UX Host Utilities 6.0 prend en charge les versions du système d'exploitation HP-UX suivantes :

* HP-UX 11iv2
* HP-UX 11iv3
